# Hey there 👋 I'm Oleh Bashtovyi

**Full-Stack Developer** — .NET / React. Based in Kyiv, Ukraine. Interested in building scalable and maintainable applications with clean code and modern technologies.

## 🚀 Currently Building

**[Learnix](https://github.com/Oleh-Bashtovyi/Learnix)** — a production-grade LMS built with .NET 8 and React 19.
Clean Architecture, CQRS via MediatR, Outbox pattern, full CI/CD to Azure. 100+ REST endpoints, ~500 unit tests, SonarCloud A-rating quality gate.

## 🔧 Other Projects

- **[HiveHub](https://github.com/Oleh-Bashtovyi/HiveHub)** — real-time multiplayer web platform. Social deduction game "Find the Spy" with SignalR-driven state synchronization.
- **[TerraForge](https://github.com/Oleh-Bashtovyi/TerraForge)** *(Thesis)* — procedural terrain generation tool in Godot 4 / C#. Voxel-based, data-driven design.
- **[DotnetVoyager](https://github.com/Rev-Enging/DotnetVoyager)** — web tool for .NET assembly static analysis and decompilation. C# / IL viewer, dependency graph visualization.
